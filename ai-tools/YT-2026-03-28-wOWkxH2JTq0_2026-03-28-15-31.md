## Краткое содержание

Видео посвящено интеграции инструмента FireCrawl в Cloud Code и E-агентов для расширения функционала парсинга веб-сайтов. Автор демонстрирует два реальных кейса использования: поиск объявлений на Авито по монолитным фундаментам в регионе Санкт-Петербурга и поиск компаний, занимающихся установкой септиков. Главный тезис — FireCrawl позволяет E-агентам безопасно парсить любые сайты, обходить блокировки, экономить токены благодаря преобразованию HTML в чистый Markdown-формат и получать структурированные данные (названия, цены, контакты, ссылки). Видео предназначено для разработчиков и предпринимателей, которые хотят автоматизировать сбор данных и конкурентный анализ с помощью AI-агентов.

## Ключевые моменты

- **FireCrawl как инструмент парсинга**: это универсальный парсер, который позволяет собирать данные с маркетплейсов (Авито, Петрович и других), социальных сетей, сайтов объявлений и любых веб-ресурсов по различным критериям — по регионам, изображениям и произвольным параметрам поиска. Инструмент обладает огромным функционалом и может быть интегрирован непосредственно в Cloud Code для взаимодействия через Telegram.

- **Первый кейс — поиск объявлений на Авито**: E-агент был попрошен найти объявления по услугам и устройствам для заливки монолитного фундамента в Санкт-Петербурге и Ленинградской области, а затем создать подробную таблицу в Obsidian со ссылками на объявления. Процесс занял всего 10 минут, несмотря на то что Авито активно борется с парсингом и ботами, блокируя доступ.

- **Второй кейс — поиск компаний-установщиков септиков**: агент искал 30 компаний, занимающихся установкой септиков для частных домов в указанных регионах, и создал таблицу с ссылками, телефонами и email-адресами. Задача была выполнена за 10 минут, результат пришел в виде аккуратной заметки в Obsidian с дополнительной информацией (специализация, регион, цены).

- **Обход блокировок сайтов**: FireCrawl позволяет обойти системы защиты и блокировки, которые устанавливают сайты против ботов (например, капчи на Авито). Это дает агентам возможность собирать полную структуру сайта и информацию одновременно, не переходя на отдельные подстраницы.

- **Экономия токенов за счет преобразования формата**: когда E-агент самостоятельно парсит сайты, он видит весь HTML-код и мусор, что сжигает огромное количество токенов. FireCrawl преобразует содержимое сайтов в чистый Markdown-формат (заголовки, текст, разделы, описания), что позволяет агенту работать с удобным и структурированным контентом без излишних данных.

- **Использование с N8N**: автор ранее интегрировал FireCrawl с N8N для создания автоматизаций, что позволяло сравнивать цены на несколько сайтах строительных гипермаркетов. Это был успешный опыт, который теперь переносится на уровень E-агентов.

- **Мониторинг изменений и отслеживание конкурентов**: с помощью FireCrawl можно настроить отслеживание изменений цен на конкурентов в течение 10 дней и более, с периодическими проверками несколько раз в час. Агент может отправить уведомление, когда появляется скидка на интересующий товар.

- **Интернет-исследования и сбор структурных данных**: E-агент может проводить комплексные интернет-исследования, собирать факты, создавать отчеты, вытаскивать структурированные данные (названия компаний, цены, email, телефоны, описания услуг).

- **Бесплатные кредиты и экономичность**: FireCrawl предоставляет около 500 бесплатных кредитов в месяц для новых пользователей, что позволяет тестировать и работать с инструментом без дополнительных затрат. Автор расходует около 430 кредитов в месяц при периодическом использовании и имеет накопленный остаток в 8178 кредитов.

- **Регистрация и получение API-ключа**: для подключения FireCrawl к Cloud Code необходимо зарегистрироваться на сайте FireCrawl, получить API-ключ в разделе API-keys, нажав Create новый ключ, скопировать его и использовать для интеграции через MCP (Model Context Protocol).

- **Интеграция через MCP**: подключение FireCrawl происходит через MCP, что является самым быстрым стартом. Нужно передать инструкцию и API-ключ E-агенту, после чего он сам настроится и получит доступ ко всем инструментам FireCrawl после перезагрузки сессии.

- **Облегченная инструкция в Obsidian**: автор создал подробную инструкцию по работе с FireCrawl, основанную на анализе 7 видеороликов с YouTube через собственного E-агента. Инструкция доступна в двух форматах: PDF и Markdown (для прямого предоставления E-агенту).

- **Использование Scraper Creators**: для сбора и анализа информации с YouTube автор использовал своего E-агента (Scraper Creators), который автоматически выгружает релевантные видео по интересующей теме и проводит их анализ.

- **Примеры задач для решения с FireCrawl**: в инструкции приведены практические примеры задач, которые можно решать с помощью инструмента, включая сравнение цен, мониторинг конкурентов, поиск контактов, сбор данных о компаниях и товарах.

- **Интеграция с Cloud Code и облачными агентами**: FireCrawl может быть добавлен к любому облачному коду (Cloud Code) и E-агентам, что позволяет использовать парсинг социальных сетей, маркетплейсов и больших веб-сайтов в едином интерфейсе Telegram.

## Инструменты и технологии

### FireCrawl
- **Описание**: универсальный инструмент для парсинга веб-сайтов, маркетплейсов, социальных сетей и любых онлайн-ресурсов. Преобразует HTML-содержимое в чистый Markdown-формат, позволяет обходить блокировки сайтов, экономит токены при работе с E-агентами.
- **Использование**: интеграция с Cloud Code и E-агентами через MCP, сбор структурированных данных (контакты, цены, описания), мониторинг изменений цен, конкурентный анализ, интернет-исследования.
- **API и кредиты**: требует регистрации и API-ключа, предоставляет 500 бесплатных кредитов в месяц для нового пользователя.

### Obsidian
- **Описание**: система заметок, используется как память для E-агентов. В видео используется для сохранения результатов парсинга в виде структурированных таблиц.
- **Использование**: автоматическое создание таблиц с найденной информацией (объявления на Авито, данные о компаниях, цены, контакты).

### Cloud Code (Cloud Clo)
- **Описание**: облачная среда кодирования, через которую осуществляется взаимодействие с E-агентами посредством Telegram.
- **Использование**: интеграция FireCrawl, управление E-агентами, автоматизация задач парсинга и анализа данных.

### N8N
- **Описание**: платформа для создания автоматизаций и интеграций между различными сервисами и инструментами.
- **Использование**: в прошлом автор использовал FireCrawl через N8N для автоматизации сравнения цен на разных сайтах. Упоминается как альтернативный способ интеграции FireCrawl.

### MCP (Model Context Protocol)
- **Описание**: протокол для интеграции инструментов и расширений в E-агентов.
- **Использование**: подключение FireCrawl к E-агентам через MCP является самым быстрым способом активации инструмента.

### Scraper Creators
- **Описание**: E-агент, созданный автором для автоматического поиска и сбора информации с YouTube.
- **Использование**: использовался для выгрузки 7 видеороликов по FireCrawl, которые затем были проанализированы для создания подробной инструкции.
