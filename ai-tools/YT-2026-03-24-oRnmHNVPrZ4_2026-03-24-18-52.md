## Краткое содержание

Видео представляет полное руководство по превращению Cloud Code в Telegram-бота, который позволяет взаимодействовать с мощным ИИ-агентом без использования терминала. Автор решает основную проблему Cloud Code — сложность терминального интерфейса, который недоступен для непрограммистов. Вместо этого создается полнофункциональный AI-помощник, работающий через удобный интерфейс Telegram с поддержкой анализа видео, генерации контента, обработки голосовых сообщений и работы с различными API. Руководство содержит пошаговые инструкции по настройке сервера, конфигурации Cloud Code, созданию Telegram-бота и подключению всех необходимых сервисов. Видео предназначено для предпринимателей, контент-мейкеров и всех, кто хочет автоматизировать свою работу с помощью AI, при минимальных затратах (подписка Pro за $17/месяц).

## Ключевые моменты

- **Проблема Cloud Code**: Большинство пользователей считают его инструментом только для программистов из-за необходимости работы через терминал, что ограничивает доступность для креативных специалистов и предпринимателей.

- **Решение через Telegram-интеграцию**: Преобразование Cloud Code в Telegram-бота позволяет использовать все его возможности через привычный мессенджер, сохраняя полный функционал AI-агента с дополнительными возможностями платформы.

- **Функциональность AI-агента**: Созданный бот способен выполнять множество задач — создание контента, анализ видео, генерация изображений, обработка голосовых сообщений, проведение исследований, работа с кодом и автоматизация процессов.

- **Экономная подписка**: Использование официальной подписки Cloud Code Pro ($17/месяц) вместо API-взаимодействия значительно дешевле и позволяет создать полнофункционального помощника без значительных расходов.

- **Реальный пример использования**: Автор демонстрирует практическое применение — создание сметы проекта с указанием цен от поставщиков и ссылок на товары, которую бот выполняет самостоятельно за 30 минут.

- **Первый шаг — покупка сервера**: Необходимо приобрести виртуальный сервер (минимальные характеристики, от $4-6/месяц) в компании вроде Vultr для размещения Cloud Code в изолированной среде, что обеспечивает безопасность и стабильность.

- **Выбор операционной системы**: Рекомендуется использовать Ubuntu 22.04 как операционную систему сервера благодаря её стабильности и совместимости с Cloud Code.

- **SSH-подключение и терминальный доступ**: Для подключения к серверу необходим SSH-ключ, который позволяет взаимодействовать с сервером через терминал (Terminus, VS Code или встроенный интерфейс провайдера).

- **Обновление системы и установка зависимостей**: Первоначальная настройка включает обновление операционной системы, установку необходимых утилит и проверку версий установленного ПО через последовательное выполнение команд.

- **Авторизация в Cloud Code**: После установки Cloud Code необходима авторизация через официальный аккаунт Anthropic, которая может быть выполнена как через браузер на сервере, так и через API-ключ, если браузера нет.

- **Выбор метода авторизации**: Рекомендуется использовать официальную подписку Anthropic вместо API-ключа для экономии денег и упрощения управления лимитами.

- **Создание Telegram-бота через BotFather**: В Telegram необходимо создать бота через официального BotFather, скопировать API-токен бота и найти свой Chat ID (через GetIDBot или UserIDInfoBot).

- **RebootPrompt.en файл**: Специальный промпт (более 1100 строк кода) служит инструкцией для Cloud Code и создаёт мост между Cloud Code и Telegram, обеспечивая взаимодействие между системами.

- **Легальность моста между сервисами**: Anthropic официально разрешает использование мостов для WhatsApp, Telegram, Discord и других мессенджеров, поощряя интеграцию функционала через удобные пользователю интерфейсы.

- **Загрузка файлов через VS Code**: Файл RebootPrompt.en загружается в папку сервера через Visual Studio Code с подключением через SSH, что удобнее и безопаснее, чем попытка вручную скопировать большой текст.

- **Взаимодействие с AI через текстовый промпт**: Вместо выполнения команд вручную можно написать единый текстовый промпт, в котором Cloud Code сам выполнит все необходимые действия по инструкции из загруженного файла.

- **Использование модели Claude Sonnet 4.6**: Для настройки рекомендуется использовать модель Claude Sonnet 4.6 (более экономная), хотя Claude Opus 4.6 обеспечивает более глубокие размышления, но требует большего расхода токенов.

- **Контроль расходования токенов**: При использовании подписки Pro видно, сколько процентов подписки потрачено на каждую операцию (полный процесс установки занимает примерно 8-17% месячной подписки).

- **Создание памяти агента**: Cloud Code может быть настроен с четырёхуровневой системой памяти, которая позволяет ему запоминать контекст и историю взаимодействий.

- **Создание дополнительных агентов (Skills)**: Через основного агента можно создавать дополнительных агентов-помощников для специализированных задач, например для анализа видео YouTube или TikTok.

- **Опции подключения дополнительных сервисов**: AI-агент может быть интегрирован с OpenAI API, Google API (Gemini), NanoNANO 2 для генерации изображений и другими сервисами по мере необходимости.

- **Система одобрения команд**: На начальном этапе Cloud Code просит одобрения для выполнения команд на сервере, что является защитой безопасности, позволяющей контролировать действия агента.

- **Исправление ошибок автоматически**: Если возникают ошибки (особенно в первые дни работы), AI-агент может самостоятельно находить их в логах и исправлять без необходимости ручного вмешательства.

- **Скиллы как расширения функционала**: Скиллы (Skills) — это дополнительные инструменты, которые расширяют возможности Cloud Code, устанавливаются как через Telegram-интерфейс, так и через терминал.

- **Практическое применение и монетизация**: После освоения процесса создания таких AI-агентов можно предоставлять эту услугу как фрилансер или компания, так как спрос на подобные решения высокий, а стоимость создания низкая.

## Инструменты и технологии

**Cloud Code** (https://anthropic.com/pricing)
- Основной инструмент для создания AI-агента на базе Claude. Используется подписка Pro за $17/месяц (примерно 1500 рублей). Позволяет работать с мощной моделью Claude напрямую без дополнительных затрат на API.

**Vultr** (https://www.vultr.com)
- Провайдер виртуальных серверов для размещения Cloud Code. Минимальный тариф от $4-6/месяц. Используется для создания изолированной среды и обеспечения безопасности данных.

**Ubuntu 22.04**
- Операционная система сервера, рекомендуемая за стабильность и совместимость с Cloud Code.

**Telegram BotFather** (https://t.me/botfather)
- Официальный инструмент Telegram для создания ботов. Позволяет создавать, управлять и получать API-токены для ботов.

**GetIDBot** (https://t.me/getidsbot)
- Telegram-бот для быстрого получения собственного Chat ID, необходимого для авторизации в кастомном боте.

**Terminus**
- Приложение для удобного подключения к серверу через SSH вместо встроенного терминала провайдера.

**Visual Studio Code** (https://code.visualstudio.com)
- IDE для работы с файлами на сервере через Remote SSH, удобнее, чем работа через встроенный терминал для загрузки файлов.

**RebootPrompt.en файл** (более 1100 строк)
- Специализированный промпт, содержащий инструкции для Cloud Code по созданию моста с Telegram. Существует на английском и русском языках.

**Claude Sonnet 4.6 и Claude Opus 4.6**
- Языковые модели для обработки запросов. Sonnet — более экономная версия, Opus — более мощная с глубокими размышлениями.

**OpenAI API**
- Дополнительный сервис для интеграции с ChatGPT функционалом, если требуется (требует отдельного API-ключа).

**Google Gemini API**
- Интеграция для анализа видео и других задач через Google API.

**Nano Banana v2** (https://www.nanobana.com)
- Сервис для генерации изображений, интегрируемый в Cloud Code.

**Obsidian**
- Приложение для ведения заметок, с которым может быть интегрирован Cloud Code для сохранения отчётов и документов.

**Notion**
- Сервис управления проектами и документами, с интеграцией для сохранения отчётов AI-агента.
