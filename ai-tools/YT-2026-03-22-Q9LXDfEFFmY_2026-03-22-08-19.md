## Краткое содержание

Видео демонстрирует революционный подход к работе с AI-агентами на примере Claude Code и сравнивает его с традиционным инструментом N8n для автоматизации. Автор показывает, как за несколько минут можно подключить сложные инструменты (Nano Banana 2, Notion, парсинг контента) к Claude боту через простые текстовые команды в Telegram, без необходимости создавать сложные N8n workflows, которые занимают часы работы. Видео предназначено для всех, кто работает с автоматизацией, AI-агентами и хочет ускорить процесс разработки без потери функциональности. Главный тезис — агенты на базе Claude код меняют весь подход к работе с автоматизацией и делают её доступнее.

## Ключевые моменты

- **Новая версия Nano Banana 2**: Вышла обновленная версия генератора изображений от Google, которая работает лучше, умнее и дешевле, чем предыдущая версия Nano Banana Pro, имеет огромное количество преимуществ перед конкурентами.

- **Экономия на API через Key AI**: Использование сервиса Key API (или Key AI) позволяет значительно снижать стоимость работы с инструментами Google, цена на Nano Banana через Key AI в несколько раз дешевле официального использования, хотя точная причина скидок неясна.

- **Сравнение старого подхода с N8n**: Раньше подключение нового инструмента требовало создания полноценного N8n workflow, что занимало 4-8 часов работы, особенно если требовалась интеграция с несколькими сервисами и использование AI-агентов.

- **Архитектура N8n workflows**: Предыдущая система подразумевала создание основного workflow с сохранением файлов в Google Документах, а затем разработку нескольких под-агентов (sub-templates), каждый из которых решал отдельную задачу (редактирование изображений, генерация, комбинирование).

- **Революционный подход с Claude Code**: Новый метод позволяет подключить Nano Banana 2 к Claude боту всего за два промта, без необходимости создания сложных N8n workflows и ручной работы.

- **Первый промт — запрос документации**: На первый запрос автор просит подключить Nano Banana 2 через Key API и предоставляет ссылку на документацию Key API, при этом сам не читал полную документацию.

- **Знание о требуемых параметрах**: Автор заранее знал, какие параметры нужны для работы: форма промта, несколько вариантов добавления изображений, выбор разрешения изображения, информация о стоимости запросов.

- **Второй промт — передача API ключа**: Во втором промте автор передает API ключ от Key API и указывает требуемые параметры: разрешение 1K по умолчанию, формат вывода PNG, соотношение сторон 16:9, усиление промтов если необходимо.

- **Автоматическое тестирование**: Claude автоматически тестирует все функции Key AI самостоятельно, о чем можно судить по логам, где видны запросы и время тестирования (например, в 18:41).

- **Сложная задача с изображениями**: После подключения автор сразу дает сложную задачу — создать логотип Claude Code, преобразовав изображение червяка в логотип Claude Cloud с правильным стилем.

- **Обработка нескольких изображений**: При отправке нескольких изображений сразу они приходят в Telegram с небольшой задержкой, но Claude успешно их обрабатывает и создает нужный результат.

- **Интеграция с N8n для парсинга**: Несмотря на преимущества Claude Code, автор все еще использует N8n для парсинга YouTube видео через Scraper Creators, так как не нашел лучшего инструмента парсинга.

- **Причины оставить N8n**: N8n используется для просмотра логов, хранения материалов, использования других LLM моделей, разгрузки Claude от перегрузки при обработке данных.

- **Интеграция Notion через MCP**: Notion подключен к Claude боту через MCP (Model Context Protocol) доступ, что позволяет боту создавать и заполнять страницы в Notion самостоятельно.

- **Система отслеживания внедрений**: В Notion создано отдельное рабочее пространство, где Claude записывает все внедренные функции и их статусы, помогая отслеживать прогресс в разработке.

- **Чтение памяти из ChatGPT и Claude**: Автор выгрузил официально сохраненную память о себе из ChatGPT и веб-версии Claude, затем передал эти файлы Claude боту для заполнения профиля.

- **Notion Skills от официального источника**: Были внедрены официальные Skills от Notion для Claude Code, которые добавили 4 скилла и 5 команд, облегчающих работу с сервисом.

- **Проблема стоимости шаблонов Notion**: Хорошие шаблоны в магазине Notion стоят 29.990 рублей или требуют ежемесячной подписки, что дорого, особенно если часто нужны новые шаблоны.

- **Метод создания шаблонов Claude**: Вместо покупки готовых шаблонов автор просто отправляет Claude боту 12-секундное видео с демонстрацией нужного шаблона и просит создать похожий в его Notion.

- **Быстрота выполнения задач**: Claude создает полноценные Notion шаблоны за 5-6 минут, включая создание папок, таблиц, свойств и описания функций.

- **Second Brain шаблон**: Бот самостоятельно создал сложный Second Brain шаблон с разделами по проектам, целям, заметкам, ресурсам и цитатам, хотя и не идеальный, но функциональный.

- **Стоимость использования**: Все эти операции выполняются на минимальной подписке Claude Code за 17 долларов в месяц, при этом автор не платит дополнительно за каждое действие бота.

- **Необходимость обновления тарифа**: Автор рассматривает переход на максимальный тариф Claude Code, так как лимиты использования бота увеличиваются в 5 раз, что позволит выполнять больше операций.

- **Использование Telegram как интерфейса**: Для работы с Claude ботом автор считает Telegram более удобным, чем терминал, хотя техически оба способа работают.

- **Инструкции и раздаточные материалы**: В закрепленном сообщении под видео находятся полные инструкции по разворачиванию Claude Code бота, описание ошибок, промпты и презентация с объяснением преимуществ Claude Code перед OpenAI.

## Инструменты и технологии

**Nano Banana 2** (от Google)
- Генератор изображений, улучшенная версия Nano Banana Pro
- Работает дешевле и качественнее предыдущих версий
- Используется для создания изображений по текстовым описаниям
- Поддерживает различные разрешения (1K, 4K и выше) и форматы (PNG, JPEG)

**Key API / Key AI**
- Сервис, предоставляющий доступ к инструментам Google со значительными скидками
- Цена использования Nano Banana через Key API в несколько раз дешевле официального API
- Требует API ключ для подключения
- Используется для экономии на затратах при работе с AI инструментами

**Claude Code (Claude 3.5 Sonnet)**
- Официальная версия Claude с интегрированными инструментами разработки
- Работает через веб-интерфейс и Telegram
- Поддерживает интеграцию с внешними сервисами через простые промты
- Стоит $17 в месяц за минимальный тариф

**Claude Code Bot (Claude Clou Bot)**
- Собственный AI агент на базе Claude Code
- Работает в Telegram для удобства
- Может самостоятельно тестировать подключенные инструменты
- Интегрирует множество сервисов без создания N8n workflows

**N8n**
- Платформа для создания сложных автоматизаций и workflows
- Используется для парсинга YouTube видео (через Scraper Creators)
- Позволяет просматривать логи и хранить материалы
- На данный момент используется как дополнительный инструмент для разгрузки Claude
- Требует 4-8 часов на создание сложного workflow

**Scraper Creators**
- Инструмент для парсинга YouTube видео
- Интегрирован в N8n workflow
- Используется для извлечения информации из видеоконтента

**Notion**
- Система управления знаниями и заметок
- Подключена к Claude боту через MCP (Model Context Protocol)
- Используется для отслеживания внедренных функций и целей
- Дорогие шаблоны (от 29.990 рублей) можно создавать через Claude вместо покупки

**Notion MCP**
- Model Context Protocol для безопасного подключения Notion к Claude
- Требует MCP ключ для подключения
- Позволяет боту создавать и редактировать страницы в Notion автоматически

**Notion Skills**
- Официальные скиллы от Notion для Claude Code
- Содержат 4 скилла и 5 команд для работы с Notion
- Расширяют возможности взаимодействия Claude с Notion

**Perplexity**
- Поисковая система для поиска шаблонов и информации
- Используется для исследования доступных шаблонов в Notion

**Google Drive**
- Облачное хранилище, планируется подключить к Claude боту
- Будет использоваться для сохранения сгенерированных файлов
- Интеграция планируется постепенно

**GitHub**
- Используется как источник официальных Skills от Notion
- Содержит инструкции и шаблоны для Claude Code
