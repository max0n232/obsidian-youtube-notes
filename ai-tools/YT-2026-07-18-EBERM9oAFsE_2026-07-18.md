## Краткое содержание
В этом видео демонстрируется интеграция n8n и Obsidian для автоматизации создания, редактирования и управления заметками. Автор объясняет, как настроить n8n для взаимодействия с файлами Markdown в Obsidian, используя SFTP для безопасной передачи данных на сервер. Также показан пример использования этой связки с AI-агентами (Claude Code, ChatGPT, Gemini, Claude) для SEO-оптимизации YouTube-видео путем анализа субтитров и генерации ключевых слов, описаний и хэштегов. Видео будет полезно тем, кто хочет автоматизировать свои процессы работы с Obsidian и активно использует AI-инструменты.

## Ключевые моменты
*   Obsidian рассматривается как текстовый редактор, хранящий заметки в виде отдельных файлов, что позволяет n8n взаимодействовать с ними через файловые операции (создание, изменение, удаление).
*   Для подключения n8n к Obsidian используется нода "Convert to File" с опцией "Convert to Text File" для создания Markdown-заметок, а затем SFTP для передачи данных.
*   Рекомендуется использовать SFTP вместо FTP для безопасной передачи файлов, так как это зашифрованный протокол.
*   Настройка SFTP-подключения требует указания хоста (IP-адреса хостинга), порта (обычно 22), имени пользователя и приватного SSH-ключа (SSH Key) для авторизации без пароля.
*   Автор использует хостинг (около $4/месяц) для синхронизации, а также интегрирует Obsidian с AI-агентами (Claude Code, Codex, Antigravity AI, Hermes) через Telegram для личного использования.
*   Автоматизация для YouTube SEO: субтитры видео (сгенерированные CapCut) отправляются в Claude Code, который затем рассылает их в три Workflow n8n, подключающие ChatGPT, Gemini и Claude.
*   Каждый AI генерирует свой вариант SEO-оптимизации (ключевые слова, описание, хэштеги), которые сохраняются в Obsidian. Затем Claude Code объединяет лучшие элементы из всех вариантов.
*   Использование SFTP ноды в n8n позволяет выполнять операции `upload` (загрузка), `rename` (переименование) и `delete` (удаление) заметок в Obsidian.
*   Шаблон синхронизации n8n-Obsidian и более 2000 пользовательских шаблонов для n8n доступны по ссылке в первом закрепленном сообщении.

## Инструменты и технологии
*   **Obsidian**: Приложение для работы с заметками, рассматривается как файловая система для взаимодействия.
*   **n8n**: Инструмент для автоматизации и интеграции, используется для управления файлами Obsidian и оркестрации AI-агентов.
*   **SFTP**: Протокол для безопасной передачи файлов, используется для загрузки/синхронизации заметок в Obsidian.
*   **Claude Code/ChatGPT/Gemini/Claude**: AI-агенты, используемые для генерации SEO-оптимизированного контента (ключевые слова, описания, хэштеги).
*   **CapCut**: Инструмент для генерации субтитров к видео.
*   **Telegram**: Мессенджер, через который автор взаимодействует со своими AI-агентами.
*   **Termius**: SSH-клиент, упоминается как способ управления SSH ключами.
*   **Yandex Disk, Google Drive, GitHub**: Упоминаются как альтернативные способы синхронизации файлов вместо SFTP.
*   **Obsidian Sync**: Платный сервис синхронизации для Obsidian.
