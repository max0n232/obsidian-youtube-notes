## Краткое содержание
Видео о MARP (Markdown Presentation Ecosystem) — инструменте генерации слайдов из Markdown. Обнаружен в вирусном твите Андрея Карпатого (~19.5M просмотров) о методах усвоения знаний. Главная идея: Claude Code + MARP skill генерирует полноценные PDF/HTML/PPTX-презентации за секунды из любых данных.

**Как работает:**
- Skill-файл в `.claude/skills/MARPSLIDE/` загружается Claude Code
- На вход: URL, README, локальный файл с данными
- Claude генерирует `.md` с CSS + Markdown разметкой
- VS Code расширение MARP рендерит live preview + экспорт в PDF/PPTX
- Заменяет Canva, PowerPoint, Gamma

**Возможности MARP:**
- Таблицы, изображения, SVG-иконки, диаграммы (QuickCharts)
- Анимации, тоглы, несколько тем (light/dark)
- Чистый Markdown → полная совместимость с Git, Obsidian, VS Code

## Применимость к нашей архитектуре

### 1. Claude Code skills — прямое применение
- Паттерн `.claude/skills/` — **именно то что мы используем**
- MARP skill кладётся в `~/.claude/skills/MARPSLIDE/` → доступен глобально
- Команда: "используй skill Marp Slides, создай презентацию из [данные]"

### 2. Отчёты из n8n данных
- n8n workflow генерирует данные (SEO audit, GSC metrics, CWV) → Claude Code + MARP skill → **готовая PDF-презентация для клиента**
- Замена ручной работы в Canva/Gamma для отчётов Studiokook

### 3. Документация архитектуры
- Вместо длинных `.md` для components → MARP-слайды для быстрого обзора
- Vault notes → MARP → share с кем-то без доступа к Obsidian

### 4. Карпатый-метод обучения
- Просить Claude отвечать MARP-слайдами при изучении сложных тем
- Визуальная структура помогает запоминанию
