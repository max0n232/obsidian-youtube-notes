## Краткое содержание

Видео посвящено созданию продвинутого Firecrawl AI-агента с использованием MCP (Model Context Protocol) сервера в n8n. Автор демонстрирует альтернативный способ интеграции Firecrawl, который позволяет обходиться без многочисленных отдельных нод для разных задач. Вместо этого используется единый MCP сервер, который может обрабатывать практически все задачи веб-скрапинга и анализа данных через интерфейс Telegram бота. Видео содержит практические примеры использования агента для анализа конкурентов, сравнения цен, поиска информации и сбора данных из интернета. Материал предназначен для предпринимателей, маркетологов и разработчиков, желающих автоматизировать процессы сбора и анализа информации из сети.

## Ключевые моменты

- **Firecrawl как универсальный инструмент**: Firecrawl — это сервис, который позволяет выполнять скрапинг сайтов, извлекать данные, искать информацию в интернете, отслеживать тренды, цены, конкурентов и отзывы. Инструмент имеет огромное количество возможностей и может использоваться бесплатно.

- **Применение для обычных пользователей**: Обычный человек может использовать Firecrawl для сбора учебных материалов, создания личной новостной ленты, сравнения цен и отзывов, подбора путешествий, создания личного каталога фильмов и отслеживания любимых блогов.

- **Возможности для маркетологов**: Маркетологи могут анализировать конкурентов, собирать тексты и цены, мониторить отзывы, исследовать ниши, создавать базы промптов, проводить SEO анализ и формировать базы лидов.

- **Применение для предпринимателей**: Для бизнесменов доступны анализ конкурентов, мониторинг партнеров, автоматизация отчетов, создание баз контента, поиск новых идей, сбор данных и создание автоматических процессов.

- **Проблема с предыдущим подходом**: В предыдущей версии агента требовалось создавать отдельные настроенные ноды для каждого типа задачи — для скрапинга, поиска информации, анализа комментариев и т.д., что делало агент громоздким и неудобным.

- **Решение через MCP сервер**: Новый подход использует MCP сервер, который позволяет обрабатывать практически все задачи Firecrawl через единый интерфейс AI-агента без необходимости создания множества отдельных компонентов.

- **Интеграция с Telegram**: Агент подключается через Telegram бот, что позволяет отправлять запросы естественным языком и получать результаты прямо в мессенджер, с дополнительным сохранением данных в таблицы.

- **Практический пример анализа конкурентов**: Агент может анализировать сайты агентств таргетированной рекламы, извлекать их уникальные предложения (оферы), описывать преимущества и составлять обзор общих предложений, встречающихся на рынке.

- **Пример сравнения цен**: Агент способен сравнивать цены на продукты (например, PlayStation 5) во множестве интернет-магазинов в конкретном городе и выводить структурированный список с ценами и источниками.

- **Поиск трендов и новостей**: Агент может находить последние новости о стартапах в сфере искусственного интеллекта, проводить глубокий анализ и предоставлять обзор актуальных тенденций.

- **Сбор данных о блогах и аккаунтах**: Система может собирать информацию о популярных блогах в социальных сетях (например, в Instagram) по определённой тематике с описанием контента и ссылками.

- **Использование Google Таблиц и DataTable**: Результаты работы агента записываются в Google Таблицы и DataTable для удобного просмотра и анализа, так как Telegram имеет ограничение на количество символов в сообщении.

- **Ценовая модель Firecrawl**: Сервис предоставляет бесплатный тариф с 500 кредитами (примерно 50 запросов), которого достаточно для периодических задач. Один запрос равен одному кредиту.

- **API ключ и MCP URL**: Для настройки агента необходимо получить API ключ в Firecrawl и скопировать URL MCP сервера из документации, затем вставить эти данные в ноду MCP Client в n8n.

- **Промпты для агента**: Система использует два промпта — пользовательский (определяющий необходимость использования MCP инструментов) и системный (содержащий подробные инструкции для работы с MCP и принятия решений). Промпты написаны на русском языке для улучшения качества ответов.

- **Выбор модели LLM**: Для агента используется Claude 3.5 Nano в качестве основной модели с Claude 3.5 в качестве резервной, так как они способны обрабатывать большое количество данных.

- **Структурированный вывод данных**: Агент преобразует сырые данные, полученные от MCP сервера, в чистый, структурированный формат с четкими ответами на русском языке, скрывая технические детали.

- **Универсальность инструмента**: Firecrawl работает со всеми сайтами — как русскоязычными, так и международными, что позволяет собирать информацию со всего интернета независимо от языка и региона.

## Инструменты и технологии

- **Firecrawl** (https://www.firecrawl.dev/) — веб-скрапинг и извлечение данных сервис, позволяет собирать информацию с сайтов, искать данные в интернете, отслеживать цены и конкурентов. Используется как основной инструмент для сбора данных в работе.

- **n8n** (https://n8n.io/) — платформа для создания автоматизации и workflow, интеграция различных сервисов через граф узлов. Используется для создания основного Firecrawl агента и организации процесса обработки данных.

- **MCP Server (Model Context Protocol)** — протокол для взаимодействия AI моделей с внешними инструментами, в данном случае с функциями Firecrawl. Позволяет единому AI агенту обрабатывать различные типы задач через стандартизированный интерфейс.

- **Claude 3.5 (Anthropic)** — AI модель, используемая в качестве основного языкового агента. Выбран для обработки сложных запросов и работы с большим объёмом данных благодаря его способности к анализу контекста.

- **Telegram Bot API** — интеграция для взаимодействия с пользователем через Telegram мессенджер, позволяет отправлять запросы и получать результаты прямо в чате.

- **Google Sheets API** — интеграция для сохранения результатов работы агента в Google Таблицы, обеспечивает удобное хранение и совместный доступ к данным.

- **DataTable** — компонент в n8n для отображения и сохранения табличных данных во внутренней базе workflow, используется как промежуточное хранилище перед экспортом в Google Sheets.

## К действию

1. **Скачать готовый шаблон**: Получить готовый настроенный шаблон Firecrawl AI-агента из первого закрепленного сообщения или описания видео и импортировать его в n8n через функцию "import from file".

2. **Зарегистрироваться на Firecrawl**: Создать учетную запись на Firecrawl (https://www.firecrawl.dev/), чтобы получить доступ к бесплатному тарифу с 500 кредитами.

3. **Получить API ключ**: В панели Firecrawl перейти в раздел API Keys, создать новый ключ и скопировать его для использования в n8n.

4. **Скопировать MCP Server URL**: В документации Firecrawl найти MCP Server URL и скопировать ссылку для подключения к n8n.

5. **Настроить MCP Client в n8n**: В ноде MCP Client вставить скопированный URL MCP сервера и полученный API ключ Firecrawl.

6. **Настроить промпты**: Убедиться, что в агенте используются правильно настроенные системный и пользовательский промпты на русском языке для улучшения качества ответов.

7. **Настроить Telegram интеграцию**: Следуя отдельным видео уроками (доступны по ссылкам в шаблоне), настроить подключение Telegram бота для взаимодействия с агентом.

8. **Настроить таблицы**: Подключить Google Sheets и DataTable ноды для сохранения результатов работы агента, так как Telegram имеет ограничение на объем текста.

9. **Тестировать агента**: Начать с простых запросов (поиск информации, сравнение цен, анализ отзывов) и постепенно усложнять задачи для проверки функциональности.

10. **Применить для бизнес-задач**: Использовать агента для анализа конкурентов, сбора лидов, мониторинга цен, поиска трендов и создания уникальных торговых предложений на основе анализа рынка.

11. **Создавать запросы на естественном языке**: Формулировать задачи простым русским языком через Telegram, позволяя AI агенту самостоятельно выбирать необходимые инструменты MCP для решения.

12. **Экспортировать и анализировать результаты**: Регулярно проверять Google Таблицы с сохраненными результатами работы агента для принятия бизнес-решений.
